# FullStack To-Do List Application

## Description  
This is a full-stack to-do list application built with Spring Boot.
Features include:
- Add new task
- Check a task
- Delete a task

#Technologies
- Spring Boot
- Thymeleaf
- Bootstrap
- MySQL
